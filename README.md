## Usage

Instal the package using `npm` (or `yarn`)

```sh
npm install --save-dev @webdevmullins/prettier-config
```

Add the `prettier` key to your `package.json`

```diff
   "license": "MIT",
   "author": "webdevmullins",
   "main": "index.js"
+  "prettier": "@webdevmullins/prettier-config",
 ```
